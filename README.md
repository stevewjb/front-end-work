# 一些本人的前端项目

## 小型博客网站(React)  工程名：my-blog

此网站使用React框架，在此网站你可以添加新的博客条目，查看现存的博客，也可以删除博客。\
此网站使用了JSON Server来模拟数据库，所以需要在本地启动JSON Server才能正常访问数据。\
你可以在本地项目路径执行以下命令来启动Server：\
npx json-server --watch data/db.json --port 8000
